# Install New Dependencies

## Framer Motion for Premium Animations

Run this command in the `attendance-dashboard` directory:

```bash
cd attendance-dashboard
npm install framer-motion
```

Or if you prefer:

```bash
cd attendance-dashboard
npm install
```

This will install Framer Motion which powers all the premium modal animations:
- Spring animations with overshoot
- Smooth entrance and exit transitions
- Backdrop blur effects
- Interactive button feedback
- Staggered content appearance

## What's Been Upgraded

All modals now have premium animations:

1. **AddCameraModal** - Add camera form with spring animations
2. **EditCameraModal** - Edit camera settings with smooth transitions
3. **ConfirmDialog** - Confirmation dialogs with bounce effects
4. **Toast** - Notification toasts with slide-in animations

Every button, input, and interactive element now has:
- Hover scale effects
- Tap feedback (bounce back)
- Smooth transitions
- Glow effects on primary actions
- Backdrop blur on overlays
